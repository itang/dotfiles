## itang's dotfiles

### Emacs

    $ git clone https://github.com/itang/dotfiles.git
    $ cp -r dotfiles/.emacs.d ~/.emacs.d
    $ emacs
    
#### about auto-complete
   
    $ cd ~/.emacs.d
    $ git clone https://github.com/itang/dotfiles.git
    $ emacs
    M - x load-file auto-complete/etc/install.el
